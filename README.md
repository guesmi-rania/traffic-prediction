# 🚦 Traffic Prediction Dashboard

A modern full-stack web application that predicts traffic levels based on the number of vehicles entered by the user.

## 🔥 Features

- 🚗 Predict traffic level (Low, Medium, High)
- 📊 Real-time traffic distribution visualization
- 🎨 Clean and modern dashboard UI
- ⚡ Fast and responsive interface

## 🛠️ Tech Stack

- Frontend: React (Vite)
- Backend: Flask (Python)
- API: REST API with Axios
- Chart: Data visualization component
- Deployment:
  - Frontend: Vercel
  - Backend: Render

## 🚀 Live Demo

Frontend: https://traffic-prediction-chi.vercel.app/
